Visión V6

ANIMA evoluciona desde una plataforma para creadores hacia un movimiento global con identidad, comunidad, IA y gamificación.

Roadmap oficial

V6 agrega una fase previa al MVP: crear cultura y confianza antes de vender masivamente.

Progreso general

0% completado
Fase 0 — Founding Era

100 embajadores

  • Crear landing oficial de ANIMA.
  • Crear formulario de Founding Ambassadors.
  • Definir beneficios de acceso vitalicio e insignia única.
  • Crear comunidad privada inicial.
  • Seleccionar contactos por país y especialidad.
  • Crear mapa mundial inicial de embajadores.
Mes 1 — MVP

Base vendible

  • Identidad visual ANIMA TSC.
  • Web App responsive + diseño premium.
  • Studio: clientes, proyectos y biblioteca.
  • Murals: cotizador, costos, materiales y PDF.
  • AI base: Director Creativo, Hunter y Productor.
  • Perfiles con XP, niveles y logros.
Mes 2 — Comunidad

Enganche

  • Crear ANIMA World.
  • Crear niveles EMBER → ANIMA + FOUNDING.
  • Crear insignias y desbloqueos.
  • Crear Creator Academy básica.
  • Crear Client Portal privado.
  • Convertir Web App en PWA instalable.
Mes 3 — Monetización

Market

  • Crear ANIMA Market.
  • Conectar Stripe o Mercado Pago.
  • Conectar Shopify API.
  • Crear comisiones del marketplace.
  • Lanzar beta cerrada con 500 usuarios.
Mes 4+ — Global

Expansión

  • Lanzamiento público en español: LATAM + España.
  • Agregar inglés como segunda ola.
  • Agregar portugués para Brasil.
  • Crear ANIMA Legacy y Constellation avanzada.
  • Crear apps nativas solo con usuarios pagando.

Stack tecnológico

Primero Web App responsive, luego PWA, finalmente apps nativas cuando haya tracción.

Frontend: React + Vite + Tailwind Backend: Supabase Hosting: Vercel IA: OpenAI + Claude Imágenes: Cloudinary Pagos: Stripe + Mercado Pago Tienda: Shopify API Correos: Resend Futuro móvil: PWA → React Native / Expo Comunidad inicial: Discord / Circle / comunidad privada propia
